# Openstartup Monorepo

This repository contains the Openstartup platform components organized in a clean module structure:
- `modules/openstartup-sdk` - SDK and core agent classes
- `modules/openstartup-agents` - individual agent packages
- `modules/openstartup-docs` - documentation Next.js site (MDX)
- `modules/openstartup-solana` - Anchor Solana program
- `modules/openstartup-marketing` - marketing landing page

## Run instructions
Examples:

- SDK: `cd modules/openstartup-sdk && pnpm install && pnpm build`
- Docs: `cd modules/openstartup-docs && pnpm install && pnpm dev`
- Solana: `cd modules/openstartup-solana && pnpm install && anchor test`
- Marketing: `cd modules/openstartup-marketing && pnpm install && pnpm dev`
